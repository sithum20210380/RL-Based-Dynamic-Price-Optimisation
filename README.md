# 🚀 Dynamic Price Optimization Using Reinforcement Learning

> An intelligent pricing system that adapts to market conditions in real-time using advanced AI techniques.

## 📊 Project Overview

This project implements a dynamic pricing optimization system for retail markets using Reinforcement Learning (RL). The system analyzes market data, predicts customer behavior, and automatically adjusts pricing strategies to maximize revenue while maintaining customer satisfaction.

### Key Features

- 🤖 Real-time price optimization using Reinforcement Learning
- 📈 Advanced market analysis and demand prediction
- 🎯 Competitor price monitoring and adaptation
- 📦 Inventory-aware pricing strategies
- 📊 Comprehensive performance metrics and visualization

## 🛠️ Technologies Used

- Python 3.x
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Q-Learning (Reinforcement Learning)

## 📋 Architecture Overview

The system consists of several key components:

1. **Data Processing Module**
   - Data loading and normalization
   - Feature engineering
   - Market condition analysis

2. **RL Agent**
   - Q-Learning implementation
   - Dynamic action space for pricing decisions
   - Reward optimization based on revenue and market conditions

3. **Market Simulation Environment**
   - Real-time market condition simulation
   - Competitor behavior modeling
   - Demand elasticity simulation

4. **Evaluation System**
   - Performance metrics calculation
   - Accuracy assessment
   - Visualization of results

## 📊 Performance Metrics

- Mean Revenue: $19.61
- Revenue Standard Deviation: $12.94
- Price Accuracy: 82.92%
- Revenue Accuracy: 86.34%
- Price Stability: 10.79

## 🚀 Getting Started

### Prerequisites

```bash
pip install numpy pandas matplotlib seaborn scikit-learn
```

### Installation

1. Clone the repository:
```bash
git clone https://github.com/sithum20210380/RL-Based-Dynamic-Price-Optimisation.git
cd dynamic-price-optimization
```

2. Install required packages:
```bash
pip install -r requirements.txt
```

3. Run the main script:
```bash
python main.py
```
